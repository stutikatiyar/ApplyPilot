# CareerOS Frontend Design System

## Color Palette

Background: #0A0E1A

Primary: #2563EB

## Design Principles

* Dark-first interface
* Career-focused branding
* Data-driven dashboard experience
* Consistent visual hierarchy
* Interactive user feedback

### Dashboard

* Header
* Search Bar
* Statistics Cards
* Recent Applications
* Analytics Chart
* AI Insights
